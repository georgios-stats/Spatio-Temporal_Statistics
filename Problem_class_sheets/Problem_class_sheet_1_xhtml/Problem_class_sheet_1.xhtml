Exercise 2.  
Show that if c 1 ( ⋅ , ⋅ ) and c 2 ( ⋅ , ⋅ ) are are covariance functions (are non-negative definite) then so are c 3 ( ⋅ , ⋅ ) =b c 1 ( ⋅ , ⋅ ) +d c 2 ( ⋅ , ⋅ ) with b,d ≥ 0 and c 4 ( ⋅ , ⋅ ) = c 1 ( ⋅ , ⋅ ) c 2 ( ⋅ , ⋅ ) .
Solution.  For all n ∈ N and a 1 ,... a n ∑ i=1 n a i ∑ j=1 n a j c 3 ( s i , s j ) = ∑ i=1 n a i ∑ j=1 n a j ( b c 1 ( s i , s j ) +d c 2 ( s i , s j ) ) = ∑ i=1 n a i ∑ j=1 n a j b c 1 ( s i , s j ) ⏟ ≥ 0 + ∑ i=1 n a i ∑ j=1 n a j d c 2 ( s i , s j ) ⏟ ≥ 0 ≥ 0 Regarding c 4 , assume independent stochastic processes ( Y s ) s ∈ S and ( X s ) s ∈ S with mean zero and covariance functions c 1 ( ⋅ , ⋅ ) and c 2 ( ⋅ , ⋅ ) correspondingly. Let stochastic processes ( Z s ) s ∈ S with Z s = Y s X s . Then Cov ( Z s , Z t ) = Cov ( Y s X s , Y t X t ) = E ( Y s X s Y t X t ) = E ( Y s Y t X s X t ) , but Y s ⊥ X s = E ( X s X t ) E ( X s X t ) = Cov ( X s , X t ) Cov ( Y s , Y t ) = c 1 ( s,t ) c 2 ( s,t ) = c 4 ( s,t ) that is c 4 ( ⋅ , ⋅ ) is a covariance function of a stochastic processes.

Theorem. (Bochner's theorem) A continuous even real-valued function c: R d → R is a covariance function of a weakly stationary random process if and only if it can be represented as c( h ) = ∫ R d exp ( i ω ⊤ h ) d F( ω ) where d F( ω ) is a symmetric positive finite measure on R d .
  • Here, we will focus on cases of the form d F( ω ) =f( ω ) d ω where f( ⋅ ) is called spectral density of c( ⋅ ) i.e. c( h ) = ∫ R d exp ( i ω ⊤ h ) f( ω ) d ω In this case, lim h → ∞ c( h ) =0

Theorem. If c( ⋅ ) is integrable, F( ⋅ ) is absolutely continuous with spectral density f( ⋅ ) of Z=( Z s ;s ∈ S ) then by Fast Fourier transformation f( ω ) = ( 1 2 π ) d ∫ R d exp ( -i ω ⊤ h ) c( h ) d h
