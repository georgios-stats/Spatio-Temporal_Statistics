Solution.      Essentially I need to show that that Var ( Z( A ) -Z( B ) ) = 1 | A | | B | ∫ x ∈ A ∫ y ∈ B γ ( x-y ) d x d y - 1 | A | | A | ∫ x ∈ A ∫ y ∈ A γ ( x-y ) d x d y - 1 | B | | B | ∫ x ∈ B ∫ y ∈ B γ ( x-y ) d x d y where I use A,B instead of v,V and x,y instead of s,s' for clarity on notation.
It is Var ( Z( A ) -Z( B ) ) = Cov ( Z( A ) -Z( B ) ,Z( A ) -Z( B ) ) = Cov ( Z( A ) ,Z( A ) ) + Cov ( Z( B ) ,Z( B ) ) -2 Cov ( Z( A ) ,Z( B ) ) = 1 | A | | A | ∫ x ∈ A ∫ y ∈ A Cov ( Z( x ) ,Z( y ) ) d x d y + 1 | B | | B | ∫ x ∈ B ∫ y ∈ B Cov ( Z( x ) ,Z( y ) ) d x d y -2 1 | A | | B | ∫ x ∈ A ∫ y ∈ B Cov ( Z( x ) ,Z( y ) ) d x d y OK, now I need to write all these Cov as γ ; I know that γ ( x-y ) = 1 2 Var ( Z( x ) -Z( y ) ) = 1 2 Var ( Z( x ) ) + 1 2 Var ( Z( y ) ) - Cov ( Z( x ) ,Z( y ) ) that is Cov ( Z( x ) ,Z( y ) ) = 1 2 Var ( Z( x ) ) + 1 2 Var ( Z( y ) ) - γ ( x-y ) Now I 'll gonna put all these in the quantity of interest, one by one 1 | A | | A | ∫ x ∈ A ∫ y ∈ A Cov ( Z( x ) ,Z( y ) ) d x d y= 1 | A | | A | ∫ x ∈ A ∫ y ∈ A 1 2 Var ( Z( x ) ) d x d y + 1 | A | | A | ∫ x ∈ A ∫ y ∈ A 1 2 Var ( Z( y ) ) d x d y - 1 | A | | A | ∫ x ∈ A ∫ y ∈ A γ ( x-y ) d x d y = 1 | A | ∫ x ∈ A Var ( Z( x ) ) d x - 1 | A | 2 ∫ x ∈ A ∫ y ∈ A γ ( x-y ) d x d y and by symmetry 1 | B | | B | ∫ x ∈ B ∫ y ∈ B Cov ( Z( x ) ,Z( y ) ) d x d y= 1 | B | ∫ x ∈ B Var ( Z( x ) ) d x - 1 | B | 2 ∫ x ∈ B ∫ y ∈ B γ ( x-y ) d x d y and finally, 1 | A | | B | ∫ x ∈ A ∫ y ∈ B Cov ( Z( x ) ,Z( y ) ) d x d y= 1 | A | | B | ∫ x ∈ A ∫ y ∈ B 1 2 Var ( Z( x ) ) d x d y + 1 | A | | B | ∫ x ∈ A ∫ y ∈ B 1 2 Var ( Z( y ) ) d x d y - 1 | A | | B | ∫ x ∈ A ∫ y ∈ B γ ( x-y ) d x d y = 1 2 1 | A | ∫ x ∈ A Var ( Z( x ) ) d x + 1 2 1 | B | ∫ x ∈ B Var ( Z( x ) ) d x - 1 | A | | B | ∫ x ∈ A ∫ y ∈ B γ ( x-y ) d x d y Putting all these together, we get the result.
