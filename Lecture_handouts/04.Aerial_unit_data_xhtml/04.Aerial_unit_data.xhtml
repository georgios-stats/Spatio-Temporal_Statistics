Part I Basic stochastic models & related concepts for model building

Note 1. Recall from Section subsec:Aerial-unit-data of “Handout enu:Handout-1:-Types: Types of spatial data” that modeling aerial unit / lattice data types involves the use of random field models with a discrete index set. Such data are collected over areal units such as pixels, census districts or tomographic bins. Often, there is a natural adjacency relation or neighborhood structure.

Note 2. This means we need to introduce new basic building models able to acceptably represent the characteristics of the underline data generating mechanisms. These as the “Discrete Random Fields”.

1 Discrete Random Fields

Note 3. We re-introduce the definition of the random field adjusting it to the aerial unit data framework.

Definition 4. A random field Z=( Z s ;s ∈ S ) on a set of indexes S taking values in Z S is a family of random variables { Z s := Z s ( ω ) ;s ∈ S , ω ∈ Ω } where each Z s ( ω ) is defined on the same probability space ( Ω , F , Pr ) and taking values in Z .

Note 5. In aerial unite data modeling, the (spatial) set of sites S , at which the process is defined, is discrete, it can be finite or infinite (e.g. S ⊆ Z d ), regular (e.g. pixels of an image) or irregular (states of a country).

Note 6. The general state space Z of the random field can be quantitative, qualitative or mixed. E.g., Z = R + in a Gamma random field, Z = N in a Poisson random field, Z ={ 0,1 } in a binary random field.

Note 7. If Z is finite or countably infinite, the (joint)distribution of Z has a PMF Pr Z ( z ) = Pr ( Z=z ) = Pr ( { Z s = z s ;s ∈ S } ) , ∀ z s ∈ Z S otherwise if Z ⊆ R d and Z continuous we will use the joint PDF.

Definition 8. The discrete set of sites S ={ s i ;i=1,...,n } is often called lattice of sites.

Notation 9. We will more often use the notation Z s instead of Z( s ) or Z i instead of Z( s i ) . Hence, since S ={ s i ;i=1,...,n } , we can consider a more convenient notation Z= ( Z s ;s ∈ S ) ⊤ = ( Z i =Z( s i ) ;i=1,...n ) ⊤ .

Notation 10. The notation i ∼ j between two sites i,j ∈ S means that “sites i and j are adjacent”.

3.1 Conditional autoregressive models (CAR)

Definition 27. Assume a random field Z=( Z s ;s ∈ S ) on a set of indexes S with values in Z . We say that Z follows a conditional autoregressive model (CAR) if the distribution of each element Z s of the random field Z is specified conditionally on the values at the neighboring sites of s .

3.1.1 Gaussian CAR

Definition 28. Gaussian CAR assumes that the local characteristics { Pr i ( z i | z S -i ) } are Gaussian distributions Z i | z S -i ∼ N ( μ i + ∑ j ≠ i b i,j ( Z j - μ j ) , κ i ) (3.1) with mean E ( Z i | Z S -i ) = μ i + ∑ j ≠ i b i,j ( Z j - μ j ) and variance Var ( Z i | Z S -i ) = κ i for i ∈ S .

Proposition 29. Let K= diag ( { κ i } ) with κ i >0 , matrix B with B i,i := [ B ] i,i =0 , and real vector μ with suitable dimentions. If Z follows a Gaussian CAR (Def 28), I-B is non-singular, and ( I-B ) -1 K>0 , then the joint distribution of Z is Z ∼ N ( μ , ( I-B ) -1 K ) . (3.2)

3.2 Simultaneus Autoregressive (SAR) models

3.2.1 Gaussian SAR

Note 32. CAR sets the AR relation, and specifies the distribution on Z which induces the distribution on E ; see 3.4. SAR does does the reverse; sets the same AR relation but it specifies the distribution on E which induces the distribution on Z this is more might be more intuitive (?).

Definition 33. Consider discrete set of sites S ={ s i ;i=1,...,n } . Consider a random field Z= ( Z s ;s ∈ S ) ⊤ = ( Z i =Z( s i ) ;i=1,...n ) ⊤ on the discrete set of indexes S with values in Z . Define Z= μ + B ˜ ( Z- μ ) +E ⟺ E=( I- B ˜ ) ( Z- μ ) Assume that matrix B ˜ is such that ( I- B ˜ ) -1 exists, and [ B ˜ ] i,i =0 . Assume that E=( E i ;i=1,...n ) is an n -dimensional Gaussian random vector E ∼ N n ( 0, Λ ) with Λ = diag ( λ 1 ,..., λ n ) whose elements are indexed by S . Then we say that Z follows a Gaussian Simultaneous Autoregressive (SAR) model.

Proposition 34. The joint distribution of Z following the SAR model in Def 33 is Z ∼ N ( μ , ( I- B ˜ ) -1 Λ ( I- B ˜ ⊤ ) -1 ) (3.5)
Proof. Z is a linear combination of Gaussians, hence if follows a Gaussian distribution. It's mean and variance are E ( Z ) = E ( ( I- B ˜ ) -1 E+ μ ) = μ , Var ( Z ) = Var ( ( I- B ˜ ) -1 E+ μ ) = ( I- B ˜ ) -1 Var ( E ) ( I- B ˜ ⊤ ) -1 = ( I- B ˜ ) -1 Λ ( I- B ˜ ⊤ ) -1

Proposition 38. Any positive-definite covariance matrix Σ can be expressed as the covariance matrix of a CAR model Σ CAR = ( I-B ) -1 K , for a unique pair of matrices B and K where ( I-B ) is non-singular and K is diagonal.
Proof. Express Σ -1 =D-R for [ D ] i,j ={ [ Σ -1 ] i,i i=j 0 i ≠ j , and [ R ] i,j ={ 0 i=j - [ Σ -1 ] i,j i ≠ j then Σ = ( D-R ) -1 = ( D( I- D -1 R ) ) -1 = ( I- D -1 R ) -1 D -1 Now define B= D -1 R and K= D -1 , and you get Σ = Σ CAR . Now regarding the uniqueness, assume there is another pair of B ˚ , and K ˚ such that Σ CAR = ( I- B ˚ ) -1 K ˚ . Then diag ( Σ -1 ) = diag ( Σ CAR -1 ) = diag ( K ˚ -1 ( I- B ˚ ) ) = diag ( K ˚ -1 ) and similarly diag ( Σ -1 ) = diag ( K -1 ) . Hence it has to be K ˚ =K because both are diagonal matrices. Then it is ( I- B ˚ ) -1 K ˚ = ( I-B ) -1 K ⟺ K ˚ =K B ˚ =B. So the representation is unique.

Proposition 39. Any positive-definite covariance matrix Σ can be expressed as the covariance matrix of a SAR model Σ SAR = ( I- B ˜ ) -1 Λ ( I- B ˜ ⊤ ) -1 for a (non-unique) pair of matrices B ˜ and Λ where ( I- B ˜ ) is non-singular, [ B ˜ ] i,i =0 , and Λ is diagonal.
Proof. Express Σ -1 =L L ⊤ where L is a lower triangular matrix with [ L ] i,i >0 . Proposition 40. Any SAR model can be written as a unique CAR model.
Proof. SAR and CAR are both Gaussian's with the same mean. SAR's variance matrix is positive definite, and hence it can be written in a unique manner as a CAR's variance matrix by Prop 38.

Proposition 41. Any CAR model can be written as a non-unique SAR model.
Proof. SAR and CAR are both Gaussian's with the same mean. CAR's variance matrix is positive definite, and hence it can be written in a non-unique manner as a SAR's variance matrix by Prop 39.

Example 42. Show that
  1. Z i and E j are independent for i ≠ j in Gaussian CAR
  2. Z i and E j are not necessarily independent for i ≠ j in Gaussian SAR
Solution.  
  1. For Gaussian CAR, Cov ( E,Z ) = Cov ( ( I-B ) Z,Z ) =( I-B ) Var ( Z ) =( I-B ) ( I-B ) -1 K=K which is a diagonal; hence Z i and E j are independent for i ≠ j .
  2. For Gaussian SAR, Cov ( Z,E ) = Cov ( ( I- B ˜ ) -1 E,E ) = ( I- B ˜ ) -1 Var ( E ) = ( I- B ˜ ) -1 Λ which is not a diagonal matrix in general; hence Z i and E j may be dependent for i ≠ j .

4 Related Random Fields

Note 43. We introduce general modeling structures of basic building models which are computationally convenient yet reasonable for use in spatial statistics models. Convenient because they aim to break a high-dimensional problem into smaller ones using conditional independence, and reasonable because they allow representation of spatial dependence as well. We introduce the Gibbs Random Fields and the Markov Random Fields. The Ising model, CAR, and SAR are just particular cases of such models.
