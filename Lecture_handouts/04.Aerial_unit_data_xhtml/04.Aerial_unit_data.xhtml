Part I Basic stochastic models & related concepts for model building

Note 1. Recall from Section subsec:Aerial-unit-data of “Handout enu:Handout-1:-Types: Types of spatial data” that modeling aerial unit / lattice data types involves the use of random field models with a discrete index set. Such data are collected over areal units such as pixels, census districts or tomographic bins. Often, there is a natural adjacency relation or neighborhood structure.

Note 2. This means we need to introduce new basic building models able to acceptably represent the characteristics of the underline data generating mechanisms. These as the “Discrete Random Fields”.

1 Discrete Random Fields

Note 3. We re-introduce the definition of the random field adjusting it to the aerial unit data framework.

Definition 4. A random field Z=( Z s ;s ∈ S ) on a set of indexes S taking values in Z S is a family of random variables { Z s := Z s ( ω ) ;s ∈ S , ω ∈ Ω } where each Z s ( ω ) is defined on the same probability space ( Ω , F , Pr ) and taking values in Z .

Note 5. In aerial unite data modeling, the (spatial) set of sites S , at which the process is defined, is discrete, it can be finite or infinite (e.g. S ⊆ Z d ), regular (e.g. pixels of an image) or irregular (states of a country).

Note 6. The general state space Z of the random field can be quantitative, qualitative or mixed. E.g., Z = R + in a Gamma random field, Z = N in a Poisson random field, Z ={ 0,1 } in a binary random field.

Note 7. If Z is finite or countably infinite, the (joint)distribution of Z has a PMF Pr Z ( z ) = Pr ( Z=z ) = Pr ( { Z s = z s ;s ∈ S } ) , ∀ z s ∈ Z S otherwise if Z ⊆ R d and Z continuous we will use the joint PDF.

Definition 8. The discrete set of sites S ={ s i ;i=1,...,n } is often called lattice of sites.

Notation 9. We will more often use the notation Z s instead of Z( s ) or Z i instead of Z( s i ) . Hence, since S ={ s i ;i=1,...,n } , we can consider a more convenient notation Z= ( Z s ;s ∈ S ) ⊤ = ( Z i =Z( s i ) ;i=1,...n ) ⊤ .

Notation 10. The notation i ∼ j between two sites i,j ∈ S means that “sites i and j are adjacent”.

The Ising model is defined by the (joint) PMF Pr Z ( z ) ∝ exp ( α ∑ i ∈ S z i + β ∑ { i,j } :i ∼ j z i z j ) , ∀ z ∈ Z S (1.1) E.g., it can model a black & white image noisy image, where S denotes the labels of the image pixels, and Z i denotes the presence of a black pixel ( Z i =1 ) or its absence ( Z i =0 ). Under Ising model, the characteristic is observed with probability Pr Z i ( z i =1 ) = exp ( α ) 1+ exp ( α ) when β =0 . The characteristic's present is encouraged in neiboghring sites when β >0 , and discouraged when β <0 .

Notation 13. We use notation, for A ⊂ S Pr A ( z A | z S \ A ) = Pr ( Z A = z A | Z S \ A = z S \ A )

Definition 14. Local characteristics of a random field Z on S with values in Z are the conditionals Pr i ( z i | z S -i ) = Pr { i } ( z { i } | z S \ { i } ) ,i ∈ S ,z ∈ Z
Example 15. The characteristics of the Ising model in (1.1) are Pr i ( z i =1| z S -i ) = exp ( α + β ∑ { i,j } :i ∼ j z j ) 1+ exp ( α + β ∑ { i,j } :i ∼ j z j )

2 Compatibility of conditional distributions


Note 16. Essentially, we attempt to answer the following question. Under what conditions a parameterized family { π i ( z i | z S -i ) ;i ∈ S } of distributions on S conditioned on z S -i can represent conditional distributions of a joint distribution Pr Z ( ⋅ ) ?

Note 17. To answer the above we need to be able to specify partially or wholly the joint and conditional distribution of Pr Z . However, an arbitrary chosen set of conditional distributions { π i ( ⋅ | ⋅ ) } is not generally compatible, and hence we need to impose conditions.

Note 22. The following theorem shows that local characteristics can determine the entire distribution in certain cases.
Theorem 23. (Besag's factorization theorem; Brook's Lemma) Let Z be a Z valued random field taking values in Z S where S ={ 1,...,n } with n ∈ N , and such as Pr Z ( z ) >0 , ∀ z ∈ Z S . Then for all Pr Z ( z ) Pr Z ( z * ) = ∏ i=1 n Pr i ( z i | z 1 ,..., z i-1 , z i+1 * ,..., z n * ) Pr i ( z i * | z 1 ,..., z i-1 , z i+1 * ,..., z n * ) , ∀ z, z * ∈ Z S (2.1)

Definition 25. Adjacency matrix N is called a matrix N with [ N ] i,j = 1 ( i ∼ j ) for some symmetric neighbourhood relation ∼ on S . It aims at spatially connecting unites i and j .

Definition 26. Proximity matrix W is called a matrix W which aims at spatially connecting unites i and j in some fashion for some symmetric neighbourhood relation ∼ on S . Usually [ W ] i,i =0

Note 27. Proximity matrix W can be an adjacency matrix, or it can be some kind of distance e.g. [ W ] i,j = dist ( i,j ) .
