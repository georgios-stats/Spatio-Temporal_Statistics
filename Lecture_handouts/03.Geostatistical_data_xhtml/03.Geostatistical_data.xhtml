1 Stochastic processes (or Random fields)

Definition 2. A stochastic process (or random field) Z=( Z s ;s ∈ S ) taking values in Z ⊆ R q , q ≥ 1 is a family of random variables { Z s := Z s ( ω ) ;s ∈ S , ω ∈ Ω } defined on the same probability space ( Ω , F , Pr ) and taking values in Z . The label s ∈ S is called site, the set S ⊆ R d is called the (spatial) set of sites at which the process is defined, and Z is called the state space of the process.

Note 3. Given a set { s 1 ,..., s n } of sites s i ∈ S the random vector ( Z( s 1 ) ,...,Z( s n ) ) ⊤ has a well-defined probability distribution that is completely determined by its joint CDF F s 1 ,..., s n ( z 1 ,..., z n ) = Pr ( Z( s 1 ) ≤ z 1 ,...,Z( s n ) ≤ z n ) Finite dimensional distributions (or fidi's) of Z is called the ensemble of all such joint CDF's with n ∈ N and { s i ∈ S } .

Note 4. To define a random field model, one must specify the joint distribution of ( Z( s 1 ) ,...,Z( s n ) ) ⊤ for all choices of n and { s i ∈ S } i=1 n in a consistent way, due to Kolmogorov thm.
Proposition 5. (Kolmogorov consistency theorem) Let Pr s 1 ,..., s n be a probability on R n with join CDF F s 1 ,..., s n for every finite collection of points s 1 ,..., s n . If F s 1 ,..., s n is symmetric w.r.t. any permutation p F p ( s 1 ) ,..., p ( s n ) ( z p ( 1 ) ,..., z p ( n ) ) = F F s 1 ,..., s n ( z 1 ,..., z n ) for all n ∈ N , { s i ∈ S } , and { z ∈ R } , and all if all permulations p are consistent in the sense lim z n → ∞ F s 1 ,..., s n ( z 1 ,..., z n ) = F s 1 ,..., s n-1 ( z 1 ,..., z n-1 ) or all n ∈ N , { s i ∈ S } , and { z ∈ R } , then there exists a random field Z whose fidi’s coincide with those in F .

Example 6. Let n ∈ N , let { f i :T → R ; i=1,...,n } be a set of constant functions, and let { Z i ∼ N ( 0,1 ) } i=1 n be a set of independent random variables. Then Z ˜ s = ∑ i=1 n Z i f i ( s ) ,s ∈ S is a well defined stochastic process as it satisfies Thm 5.

Definition 20. A random field Z= ( Z s ) s ∈ R d is strongly stationary if for all finite sets consisting of s 1 ,..., s n ∈ R d , n ∈ N , for all k 1 ,..., k n ∈ R , and for all h ∈ R d Pr ( Z( s 1 +h ) ≤ k 1 ,...,Z( s n +h ) ≤ k n ) = Pr ( Z( s 1 ) ≤ k 1 ,...,Z( s n ) ≤ k n )

Definition 22. A random field Z= ( Z s ) s ∈ R d is weakly stationary (or second order stationary) if, for all s,s' ∈ R d ,
  1. E ( Z s 2 ) < ∞ (finite)
  2. E ( Z s ) =m (constant)
  3. Cov ( Z s , Z s' ) =c( s'-s ) for some even function c: R d → R (lag dependency)

Definition 23. Weakly (or second order) stationary covariance function is called the c.f. of a weakly stationary stochastic process.

6 Covariogram

Note 24. The definition of the covariogram function requires the random field to be weakly stationary.

Definition 25. Let Z= ( Z s ) s ∈ R d be a weakly stationary random field. The covariogram function of Z= ( Z s ) s ∈ R d is defined by c: R d → R with c( h ) = Cov ( Z s , Z s+h ) , ∀ s ∈ R d .

Proposition 27. If c: R d → R is the covariogram of a weakly stationary random field Z= ( Z s ) s ∈ R d then:
  1. c( 0 ) ≥ 0
  2. c( h ) =c( -h ) for all h ∈ R d
  3. | c( h ) | ≤ c( 0 ) = Var ( Z s ) for all h ∈ R d
  4. c( ⋅ ) is semi-positive definite; i.e. for all n ∈ N , a ∈ R n , and { s 1 ,..., s n } ⊆ S ∑ i=1 n ∑ j=1 n a i a j c( s i - s j ) ≥ 0

7 Intrinsic stationarity

Note 32. Getting greedier, we can further weaken the weak stationarity by considering lag dependent variance in the increments with purpose to be able to use more inclusive models; Def 22 implies that Var ( Z s+h - Z s ) = Var ( Z s+h ) + Var ( Z s ) -2 Coc ( Z s+h , Z s ) =2c( 0 ) -2c( h ) .

Definition 33. A random field Z= ( Z s ) s ∈ R d is intrinsically stationary if, for all h ∈ R d , ( Z s+h - Z s ) s ∈ R d is weakly stationary; i.e.
  1. E ( Z s+h - Z s ) 2 < ∞
  2. E ( Z s+h - Z s ) =m (constant)
  3. Var ( Z s+h - Z s ) =2 γ ( h ) for some function γ : R d → R (lag dependent)

Definition 34. Intrinsically stationary covariance function is called the c.f. of an intrinsically stationary stochastic process.

The following covariance function is not weakly but intrinsically stationary c( s,t ) = 1 2 ( ∥ s ∥ 2H + ∥ t ∥ 2H - ∥ t-s ∥ 2H ) ,H ∈ ( 0,1 ) because for h ∈ R d c( s,s+h ) = 1 2 ( ∥ s ∥ 2H + ∥ s+h ∥ 2H - ∥ h ∥ 2H ) and Var ( Z s - Z s+h ) = Var ( Z s ) + Var ( Z s+h ) -2 Cov ( Z s , Z s+h ) = 1 2 ∥ h ∥ 2H

8 (Semi) variogram

Note 36. The definition of the semi-variogram function requires the random field to be intrinsic stationarity; which is weaker assumption than weak stationary required by covariogram.
Definition 37. Let Z= ( Z s ) s ∈ R d be intrinsically stationary. The semi-variogram of Z is defined by γ : R d → R with γ ( h ) = 1 2 Var ( Z s+h - Z s ) , ∀ s ∈ R d

Definition 38. Variogram of an intrinsically stationary random field is called the quantity 2 γ ( h ) .

Note 39. Let Z= ( Z s ) s ∈ R d be weakly stationary with covariogram c( ⋅ ) . Then Z is intrinsic stationary with semi-variogram γ ( h ) =c( 0 ) -c( h ) , ∀ h ∈ R d
