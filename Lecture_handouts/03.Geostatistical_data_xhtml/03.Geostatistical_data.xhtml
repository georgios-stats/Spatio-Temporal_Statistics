1 Stochastic processes (or Random fields)

Definition 2. A stochastic process (or random field) Z=( Z s ;s ∈ S ) taking values in Z ⊆ R q , q ≥ 1 is a family of random variables { Z s := Z s ( ω ) ;s ∈ S , ω ∈ Ω } defined on the same probability space ( Ω , F , Pr ) and taking values in Z . The label s ∈ S is called site, the set S ⊆ R d is called the (spatial) set of sites at which the process is defined, and Z is called the state space of the process.

Note 3. Given a set { s 1 ,..., s n } of sites, with s i ∈ S , the random vector ( Z( s 1 ) ,...,Z( s n ) ) ⊤ has a well-defined probability distribution that is completely determined by its joint CDF F s 1 ,..., s n ( z 1 ,..., z n ) = Pr ( Z( s 1 ) ≤ z 1 ,...,Z( s n ) ≤ z n ) Finite dimensional distributions (or fidi's) of Z is called the ensemble of all such joint CDF's with n ∈ N and { s i ∈ S } .

Note 4. According to Kolmogorov Thm 5, to define a random field model, one must specify the joint distribution of ( Z( s 1 ) ,...,Z( s n ) ) ⊤ for all of n and all { s i ∈ S } i=1 n in a consistent way.
Proposition 5. (Kolmogorov consistency theorem) Let Pr s 1 ,..., s n be a probability on R n with join CDF F s 1 ,..., s n for every finite collection of points s 1 ,..., s n . If F s 1 ,..., s n is symmetric w.r.t. any permutation p F p ( s 1 ) ,..., p ( s n ) ( z p ( 1 ) ,..., z p ( n ) ) = F s 1 ,..., s n ( z 1 ,..., z n ) for all n ∈ N , { s i ∈ S } , and { z ∈ R } , and all if all permulations p are consistent in the sense lim z n → ∞ F s 1 ,..., s n ( z 1 ,..., z n ) = F s 1 ,..., s n-1 ( z 1 ,..., z n-1 ) or all n ∈ N , { s i ∈ S } , and { z ∈ R } , then there exists a random field Z whose fidi’s coincide with those in F .

Example 6. Let n ∈ N , let { f i :T → R ; i=1,...,n } be a set of constant functions, and let { Z i ∼ N ( 0,1 ) } i=1 n be a set of independent random variables. Then Z ˜ s = ∑ i=1 n Z i f i ( s ) ,s ∈ S (1.1) is a well defined stochastic process as it satisfies Thm 5.

Definition 23. A random field Z= ( Z s ) s ∈ R d is weakly stationary (or second order stationary) if, for all s,s' ∈ R d ,
  1. E ( Z s 2 ) < ∞ (finite)
  2. E ( Z s ) =m (constant)

Definition 24. Weakly (or second order) stationary covariance function is called the c.f. of a weakly stationary stochastic process.

6 Covariogram

Note 25. The definition of the covariogram function requires the random field to be weakly stationary.

Definition 26. Let Z= ( Z s ) s ∈ R d be a weakly stationary random field. The covariogram function of Z= ( Z s ) s ∈ R d is defined by c: R d → R with c( h ) = Cov ( Z s , Z s+h ) , ∀ s ∈ R d .

Proposition 28. If c: R d → R is the covariogram of a weakly stationary random field Z= ( Z s ) s ∈ R d then:
  1. c( 0 ) ≥ 0
  2. c( h ) =c( -h ) for all h ∈ R d
  3. | c( h ) | ≤ c( 0 ) = Var ( Z s ) for all h ∈ R d
  4. c( ⋅ ) is semi-positive definite; i.e. for all n ∈ N , a ∈ R n , and { s 1 ,..., s n } ⊆ S ∑ i=1 n ∑ j=1 n a i a j c( s i - s j ) ≥ 0

Nugget Effect
Note 46. Nugget effect is the semivariagram's limiting value σ ϵ 2 = lim ∥ h ∥ → 0 γ ( h ) In particular when σ ϵ 2 ≠ 0 .

Note 47. Nugget effect σ ϵ 2 ≠ 0 may expected or assumed to appear due to (1) measurement errors (e.g., if we collect repeated measurements at the same location s ) or (2) due to some microscale variation causing discontinuity in the origin that cannot be detected from the data i.e. the spatial gaps because we collect a finite set of measurements at spatial locations. Hence theoretically, we could consider a more detailed decomposition σ ϵ 2 = σ MS 2 + σ MS 2 where σ MS 2 refers to the microscale and σ MS 2 refers to the measurement error; however (my experience) this is non-identifiable.

Note 48. For a continuous processes Z= ( Z s ) s ∈ R d , it is expected lim ∥ h ∥ → 0 E ( Z s+h - Z s ) 2 =0 which is equivalent to a continuous semivariagram γ ( h ) for all h , and in particular, lim ∥ h ∥ → 0 γ ( h ) = γ ( 0 ) =0 , because γ ( 0 ) =0 . However, when modeling a real problem we may need to consider (or it may appear from the data) that γ ( h ) should have a discontinuity lim ∥ h ∥ → 0 γ ( h ) = σ ϵ 2 ≠ 0 .

Note 49. Nugget effect is often mathematically described by considering a decomposition ; Z( s ) =Y( s ) + ϵ ( s ) (8.2) where Y can be a continuous stationary process with γ Y ( ⋅ ) , and ϵ can be a process (called errors-in-variables model) with (nugget) semivariogram γ ϵ ( h ) = σ ϵ 2 . In this case, γ Z ( h ) = γ Y ( h ) + γ ϵ ( h ) → ∥ h ∥ → 0 σ ϵ 2

Sill
Definition 50. Sill is the variogram's limiting value lim ∥ h ∥ → ∞ γ ( h ) .

Note 51. For weakly stationary processes the sill is always finite. However, for intrinsic processes, the sill may be infinite.
Partial sill
Definition 52. Partial sill is lim ∥ h ∥ → ∞ γ ( h ) - lim ∥ h ∥ → 0 γ ( h ) which takes into account the nugget.
Range
Range is the distance at which the semivariogram reaches the Sill; it can be infinite.

10 Anisotropy

Note 60. Dependence between Z( s ) and Z( s+h ) is a function of both the magnitude and the direction of separation h . This can be caused by the underlying physical process evolving differently in space (e.g., vertical and horizontal axes).
Definition 61. The variogram γ : R d → R is anisotropic if there are h 1 and h 2 with same length ∥ h 1 ∥ =∥ h 2 ∥ but different direction h 1 /∥ h 1 ∥ ≠ h 2 /∥ h 2 ∥ that produce different variograms γ ( h 1 ) ≠ γ ( h 2 ) .

Definition 62. The intrinsically stationary process ( Z s ) s ∈ R d is anisotropic if its variogram is anisotropic.

Definition 63. The covariogram c: R d → R is anisotropic if there are h 1 and h 2 with same length ∥ h 1 ∥ =∥ h 2 ∥ but different direction h 1 /∥ h 1 ∥ ≠ h 2 /∥ h 2 ∥ that produce different covariogram c( h 1 ) ≠ c( h 2 ) .

Definition 64. The weakly stationary process ( Z s ) s ∈ R d is anisotropic if its covariogram is anisotropic.

Change of support effect

Note 202. [Change of support effect] Consider the case that domain S is partitioned into volumes V which are partitioned into smaller volumes v . Assume there are available samples at “point” locations s each of them lies to the center of one of the smaller volumes v . Making the assumption that the sampled value at each point location s is extended to each area of influence v implies that the distribution of average values of the blocks is the same as the distribution of the values at the sample points. However from (16.4), we see that this is not true; in fact the distribution of the values for a support v is narrower than the distribution of point values because the variance σ 2 ( s|v ) of the points in v generally is not negligible; i.e. σ 2 ( s|V ) - σ 2 ( v|V ) = σ 2 ( s|v ) >0 .
