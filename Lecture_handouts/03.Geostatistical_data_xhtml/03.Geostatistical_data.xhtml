1 Stochastic processes (or Random fields)

Definition 2. A stochastic process (or random field) Z=( Z s ;s ∈ S ) taking values in Z ⊆ R q , q ≥ 1 is a family of random variables { Z s := Z s ( ω ) ;s ∈ S , ω ∈ Ω } defined on the same probability space ( Ω , F , Pr ) and taking values in Z . The label s ∈ S is called site, the set S ⊆ R d is called the (spatial) set of sites at which the process is defined, and Z is called the state space of the process.

Note 3. Given a set { s 1 ,..., s n } of sites, with s i ∈ S , the random vector ( Z( s 1 ) ,...,Z( s n ) ) ⊤ has a well-defined probability distribution that is completely determined by its joint CDF F s 1 ,..., s n ( z 1 ,..., z n ) = Pr ( Z( s 1 ) ≤ z 1 ,...,Z( s n ) ≤ z n ) Finite dimensional distributions (or fidi's) of Z is called the ensemble of all such joint CDF's with n ∈ N and { s i ∈ S } .

Note 4. According to Kolmogorov Thm 5, to define a random field model, one must specify the joint distribution of ( Z( s 1 ) ,...,Z( s n ) ) ⊤ for all of n and all { s i ∈ S } i=1 n in a consistent way.
Proposition 5. (Kolmogorov consistency theorem) Let Pr s 1 ,..., s n be a probability on R n with join CDF F s 1 ,..., s n for every finite collection of points s 1 ,..., s n . If F s 1 ,..., s n is symmetric w.r.t. any permutation p F p ( s 1 ) ,..., p ( s n ) ( z p ( 1 ) ,..., z p ( n ) ) = F s 1 ,..., s n ( z 1 ,..., z n ) for all n ∈ N , { s i ∈ S } , and { z ∈ R } , and all if all permulations p are consistent in the sense lim z n → ∞ F s 1 ,..., s n ( z 1 ,..., z n ) = F s 1 ,..., s n-1 ( z 1 ,..., z n-1 ) or all n ∈ N , { s i ∈ S } , and { z ∈ R } , then there exists a random field Z whose fidi’s coincide with those in F .

Example 6. Let n ∈ N , let { f i :T → R ; i=1,...,n } be a set of constant functions, and let { Z i ∼ N ( 0,1 ) } i=1 n be a set of independent random variables. Then Z ˜ s = ∑ i=1 n Z i f i ( s ) ,s ∈ S (1.1) is a well defined stochastic process as it satisfies Thm 5.

Proposition 15. A GP Z=( Z s ;s ∈ S ) is fully characterized by its mean function μ :S → R with μ ( s ) = E ( Z s ) , and its covariance function with c( s,s' ) = Cov ( Z s , Z s' ) .

Example 17. When using the GP as a model we may need to parameterize its parameters. Definition 21. A random field Z= ( Z s ) s ∈ R d is strongly stationary if for all finite sets consisting of s 1 ,..., s n ∈ R d , n ∈ N , for all k 1 ,..., k n ∈ R , and for all h ∈ R d Pr ( Z( s 1 +h ) ≤ k 1 ,...,Z( s n +h ) ≤ k n ) = Pr ( Z( s 1 ) ≤ k 1 ,...,Z( s n ) ≤ k n )

5 Weak stationarity (or second order stationarity)

Definition 23. A random field Z= ( Z s ) s ∈ R d is weakly stationary (or second order stationary) if, for all s,s' ∈ R d ,
  1. E ( Z s 2 ) < ∞ (finite)
  2. E ( Z s ) =m (constant)
  3. Cov ( Z s , Z s' ) =c( s'-s ) for some even function c: R d → R (lag dependency)

Definition 24. Weakly (or second order) stationary covariance function is called the c.f. of a weakly stationary stochastic process.

6 Covariogram

Note 25. The definition of the covariogram function requires the random field to be weakly stationary.

Definition 26. Let Z= ( Z s ) s ∈ R d be a weakly stationary random field. The covariogram function of Z= ( Z s ) s ∈ R d is defined by c: R d → R with c( h ) = Cov ( Z s , Z s+h ) , ∀ s ∈ R d .

Example 27. For the Gaussian c.f. c( s,t ) = σ 2 exp ( - β ∥ s-t ∥ 2 2 ) in (Ex. 17(2.)), we may denote just c( h ) =c( s,s+h ) = σ 2 exp ( - β ∥ h ∥ 2 2 ) (6.1) Observe that, in Figs 3.1 &3.2, the smaller the β , the smoother the realization (aka slower changes). One way to justify this observation is to think that smaller values of β essentially bring the points closer by re-scaling spatial lags h in the c.f.

Proposition 28. If c: R d → R is the covariogram of a weakly stationary random field Z= ( Z s ) s ∈ R d then:
  1. c( 0 ) ≥ 0
  2. c( h ) =c( -h ) for all h ∈ R d
  3. | c( h ) | ≤ c( 0 ) = Var ( Z s ) for all h ∈ R d
  4. c( ⋅ ) is semi-positive definite; i.e. for all n ∈ N , a ∈ R n , and { s 1 ,..., s n } ⊆ S ∑ i=1 n ∑ j=1 n a i a j c( s i - s j ) ≥ 0

Note 29. The following helps in the specification of cavariograms by considering properties of characteristic functions.

Theorem 30. (Bochner's theorem) A continuous even real-valued function c: R d → R is a covariance function of a weakly stationary random process if and only if it can be represented as c( h ) = ∫ R d exp ( i ω ⊤ h ) d F( ω ) where d F( ω ) is a symmetric positive finite measure on R d .
  • Here, we will focus on cases of the form d F( ω ) =f( ω ) d ω where f( ⋅ ) is called spectral density of c( ⋅ ) i.e. c( h ) = ∫ R d exp ( i ω ⊤ h ) f( ω ) d ω In this case, lim h → ∞ c( h ) =0

Theorem 31. If c( ⋅ ) is integrable, F( ⋅ ) is absolutely continuous with spectral density f( ⋅ ) of Z=( Z s ;s ∈ S ) then by Fast Fourier transformation f( ω ) = ( 1 2 π ) d ∫ R d exp ( -i ω ⊤ h ) c( h ) d h
Example 32. Consider the Gaussian c.f. c( h ) = σ 2 exp ( - β ∥ h ∥ 2 2 ) for σ 2 , β >0 and h ∈ R d . Then the spectral density from Thm 30 is f( ω ) = ( 1 2 π ) d ∫ R d exp ( -i ω ⊤ h ) σ 2 exp ( - β ∥ h ∥ 2 2 ) d h = σ 2 ( 1 2 π ) d ∏ j=1 d ∫ R exp ( -i ω j h j - β h j 2 ) d h = σ 2 ( 1 2 π ) d ∏ j=1 d ∫ R exp ( - β ( h j -( -i ω /( 2 β ) ) ) 2 ) d h j = σ 2 ( 1 4 π β ) d/2 exp ( - ∥ ω ∥ 2 2 /( 4 β ) ) i.e. of a Gaussian form.

7 Intrinsic stationarity

Note 33. Getting greedier, we can further weaken the weak stationarity by considering lag dependent variance in the increments with purpose to be able to use more inclusive models; Def 23 implies that Var ( Z s+h - Z s ) = Var ( Z s+h ) + Var ( Z s ) -2 Cov ( Z s+h , Z s ) =2c( 0 ) -2c( h ) .

Definition 34. A random field Z= ( Z s ) s ∈ R d is intrinsically stationary if, for all h ∈ R d , ( Z s+h - Z s ) s ∈ R d is weakly stationary; i.e.
  1. E ( Z s+h - Z s ) 2 < ∞
  2. E ( Z s+h - Z s ) =m (constant)
  3. Var ( Z s+h - Z s ) =2 γ ( h ) for some function γ : R d → R (lag dependent)

Definition 35. Intrinsically stationary covariance function is called the c.f. of an intrinsically stationary stochastic process.

Example 36. The following covariance function is not weakly but intrinsically stationary c( s,t ) = 1 2 ( ∥ s ∥ 2H + ∥ t ∥ 2H - ∥ t-s ∥ 2H ) ,H ∈ ( 0,1 ) because for h ∈ R d c( s,s+h ) = 1 2 ( ∥ s ∥ 2H + ∥ s+h ∥ 2H - ∥ h ∥ 2H ) and Var ( Z s - Z s+h ) = Var ( Z s ) + Var ( Z s+h ) -2 Cov ( Z s , Z s+h ) = 1 2 ∥ h ∥ 2H

8 (Semi) variogram

Note 37. The definition of the semi-variogram function requires the random field to be intrinsic stationarity; which is weaker assumption than weak stationary required by covariogram.
Definition 38. Let Z= ( Z s ) s ∈ R d be intrinsically stationary. The semi-variogram of Z is defined by γ : R d → R with γ ( h ) = 1 2 Var ( Z s+h - Z s ) , ∀ s ∈ R d

Definition 39. Variogram of an intrinsically stationary random field is called the quantity 2 γ ( h ) .

Note 40. Let Z= ( Z s ) s ∈ R d be weakly stationary with covariogram c( ⋅ ) . Then Z is intrinsic stationary with semi-variogram γ ( h ) =c( 0 ) -c( h ) , ∀ h ∈ R d (8.1)
Example 41. For the Gaussian covariance function (Ex. 27) the semi-variogram is γ ( h ) =c( 0 ) -c( h ) = σ 2 ( 1- exp ( - β ∥ h ∥ 2 2 ) )


Proposition 42. Properties of semi-variogrames. Let Z= ( Z s ) s ∈ R d be an intrinsically stationary process.
  1. It is γ ( h ) = γ ( -h ) , γ ( h ) ≥ 0 , and γ ( 0 ) =0
  2. Semi-variogram is conditionally negative definite (c.n.d.): for all a ∈ R n s.t. ∑ i=1 n a i =0 , and for all ∀ { s 1 ,..., s n } ⊆ S ∑ i=1 n ∑ j=1 n a i a j γ ( s i - s j ) ≤ 0
  3. If γ ( h ) is a semi-variogram, and A is a linear transformation in R d then γ ˜ ( h ) = γ ( Ah ) is a semi-variogram too.
  4. The following functions are semi-variograms
    1. γ ( ⋅ ) = ∑ i=1 n a i γ i ( ⋅ ) , if a i ≥ 0 , and { γ i ( ⋅ ) } are semi-variograms
    2. γ ( ⋅ ) = ∫ γ u ( ⋅ ) d F( u ) , if γ u ( ⋅ ) is a semi-variogram parametrized by u ∼ F
    3. γ ( ⋅ ) = lim n → ∞ γ n ( ⋅ ) if γ n ( ⋅ ) is semi-variogram and the limit exists
  5. Consider intrinsically stationary stochastic processes Y= ( Y s ) s ∈ R d and E= ( E s ) s ∈ R d where Y and E are independent each other. Let Z s = Y s + E s . Then γ Z ( h ) = γ Y ( h ) + γ E ( h )

Note 43. A semivariogram tends to be an increasing function of the lag ∥ h ∥ . Recall in weakly stationary processes, γ ( h ) =c( 0 ) -c( h ) where common logic suggests that c( h ) is decreases with ∥ h ∥ .

Note 44. If γ ( h ) is a positive constant for all lags h ≠ 0 , then Z( s 1 ) and Z( s 2 ) are uncorrelated regardless of how close s 1 and s 2 are; and Z= ( Z s ) s ∈ R d is often called white noise.

Note 45. Conversely, a non zero slope of the variogram indicates structure.
Nugget Effect
Note 46. Nugget effect is the semivariagram's limiting value σ ϵ 2 = lim ∥ h ∥ → 0 γ ( h ) In particular when σ ϵ 2 ≠ 0 .

Note 47. Nugget effect σ ϵ 2 ≠ 0 may expected or assumed to appear due to (1) measurement errors (e.g., if we collect repeated measurements at the same location s ) or (2) due to some microscale variation causing discontinuity in the origin that cannot be detected from the data i.e. the spatial gaps because we collect a finite set of measurements at spatial locations. Hence theoretically, we could consider a more detailed decomposition σ ϵ 2 = σ MS 2 + σ MS 2 where σ MS 2 refers to the microscale and σ MS 2 refers to the measurement error; however (my experience) this is non-identifiable.

Note 48. For a continuous processes Z= ( Z s ) s ∈ R d , it is expected lim ∥ h ∥ → 0 E ( Z s+h - Z s ) 2 =0 which is equivalent to a continuous semivariagram γ ( h ) for all h , and in particular, lim ∥ h ∥ → 0 γ ( h ) = γ ( 0 ) =0 , because γ ( 0 ) =0 . However, when modeling a real problem we may need to consider (or it may appear from the data) that γ ( h ) should have a discontinuity lim ∥ h ∥ → 0 γ ( h ) = σ ϵ 2 ≠ 0 .

Note 49. Nugget effect is often mathematically described by considering a decomposition ; Z( s ) =Y( s ) + ϵ ( s ) (8.2) where Y can be a continuous stationary process with γ Y ( ⋅ ) , and ϵ can be a process (called errors-in-variables model) with (nugget) semivariogram γ ϵ ( h ) = σ ϵ 2 1 { 0 } ( h ) . In this case, γ Z ( h ) = γ Y ( h ) + γ ϵ ( h ) → ∥ h ∥ → 0 σ ϵ 2

Sill
Definition 50. Sill is the variogram's limiting value lim ∥ h ∥ → ∞ γ ( h ) .

Note 51. For weakly stationary processes the sill is always finite. However, for intrinsic processes, the sill may be infinite.
Partial sill
Definition 52. Partial sill is lim ∥ h ∥ → ∞ γ ( h ) - lim ∥ h ∥ → 0 γ ( h ) which takes into account the nugget.
Range
Range is the distance at which the semivariogram reaches the Sill; it can be infinite.
Other

Note 53. An abrupt change in slope indicates the passage to a different structuration of the values in space. This is often modeled via decompoditions of processes with different semivariograms as in (8.2).

9 Isotropy

Note 54. Isotropy as a notion imposes the assumption of “rotation invariance” in the stochastic process.

Definition 55. An intrinsic stochastic process ( Z s ) s ∈ R d is isotropic iff ∀ s,t ∈ S , 1 2 Var ( Z s - Z t ) = γ ( ∥ t-s ∥ ) , for some function  γ : R + → R . (9.1)

Definition 56. Isotropic semi-variogram γ : R d → R is the semi-variogram of the isotropic stochastic process. (sometimes for simplicity of notation we use γ : R + → R with γ ( ∥ h ∥ ) = 1 2 Var ( Z s - Z s-h ) .

Definition 57. Isotropic covariance function C: S × S → R is called the covariance function satisfying (9.1).

Definition 58. Isotropic covariogram c: R d → R of a weakly stationary process is the covariogram associated to an isotropic semi-variogram (sometimes for simplicity of notation we use c: R + → R with c( ∥ h ∥ ) from (9.1)).

Note 59. Semi-variogrames are recovered from γ ( h ) =c( 0 ) -c( h ) .

9.1.1 Nugget-effect 

For σ 2 >0 , c( h ) = σ 2 1 {0} ( ∥ h ∥ ) . It is associate to white noise. It is used to model a discontinuity in the origin of the covariogram / sem-variogram.

9.1.2 Matern c.f.  

For σ 2 >0 , φ >0 , and ν ≥ 0 c( h ) = σ 2 2 1- ν Γ ( ν ) ( ∥ h ∥ φ ) ν K ν ( ∥ h ∥ φ ) Parameter ν controls the variogram's regularity at 0 which in turn controls the quadratic mean (q.m.) regularity of the associated process. For ν =1/2 , we get the exponential c.f., c( h ) = σ 2 exp ( - 1 φ ∥ h ∥ 1 ) which is not differentiable at h=0 , while for ν → ∞ , we get the Gaussian c.f. c( h ) = σ 2 exp ( - 1 φ ∥ h ∥ 2 2 ) which is infinite differentiable. φ is a range parameter, and σ 2 is the (partial) sill parameter.

9.1.3 Spherical c.f.

 For σ 2 >0 and φ >0 c( h ) ={ σ 2 ( 1- 3 2 ∥ h ∥ 1 φ + 1 2 ( ∥ h ∥ 1 φ ) 3 ) ∥ h ∥ 1 ≤ φ σ 2 ∥ h ∥ 1 > φ The c.f. starts from its maximum value b , then steadily decreases, and finally vanishes when its range φ is attained. φ is a range parameter, and σ 2 is the (partial) sill parameter.

11.1 Non-parametric estimation

Note 60. Smoothed Matheron estimator γ ˆ ( ⋅ ) of semi-variogram γ ( ⋅ ) is γ ˆ M ( h ) = 1 2| N r ( h ) | ∑ ∀ ( s,t ) ∈ N r ( h ) ( Z( s ) -Z( t ) ) 2 (11.1) where N r ( h ) ={ ( s,t ) ∈ S 2 :t-s ∈ B r ( h ) } contains all the pairs of spatial points points whole difference is in a ball B r ( h ) centered at h with radius r>0 . – γ ˆ M considers all pairs of observations that are `approximately' h apart and averages out.

Note 61. The choice of ε is an art, and a trade-off between variance and bias, similar to the bin length in histograms.

11.2 Parametric estimation

Note 62. Smoothed Matheron estimator (11.1) does not necessarily satisfies semivariogram properties, such as negative definiteness. To address this we use a parametric family of appropriate semi-variogram functions and tune them against data.

Note 63. A parametric family of semivariograms is the Matern semivariogram model γ ( h ) = σ 2 ( 1- 2 1- ν Γ ( ν ) ( ∥ h ∥ α ) ν K ν ( ∥ h ∥ α ) ) with tunable parameters: the `variance' σ 2 >0 , `range' α >0 , shape ν >0 . Here, K ν ( ⋅ ) denotes the modified Bessel function of the second kind and order ν .
