Exercise 1. ( ★ )
Let Z= ( Z s ) s ∈ R d be an intrinsically stationary stochastic process, and let γ : R d → R be its semivariogram. Assume a ∈ R n s.t. ∑ i=1 n a i =0 .
  1. Let a ∈ R n be a vector of constants. Show that Var ( ∑ i=1 n a i Z( s i ) ) = ∑ i=1 n ∑ j=1 n a i a j c Y ( s i , s j ) where c Y ( s,t ) = E ( Y( s ) Y( t ) ) , and Y s = Z s - Z 0 .
  2. Show that c Y ( s,t ) = γ ( s ) + γ ( t ) - γ ( s-t )
  3. Show that for all ∀ { s 1 ,..., s n } ⊆ S it is ∑ i=1 n ∑ j=1 n a i a j γ ( s i - s j ) ≤ 0
Solution. Assume 0 ∈ S , and a random variable Z( 0 ) . Let Y s = Z s - Z 0 .
  1. It is Var ( ∑ i=1 n a i Z( s i ) ) = Var ( ∑ i=1 n a i Z( s i ) - ∑ i=1 n a i ⏞ 0= Z( 0 ) ) = Var ( ∑ i=1 n a i Y( s i ) ) = ∑ i=1 n ∑ j=1 n a i a j E ( Y( s i ) Y( s j ) ) = c Y ( s,t )
  2. For E ( Y( s i ) ) =0 it is γ ( s-t ) = 1 2 E ( Z( s ) -Z( 0 ) +Z( t ) -Z( 0 ) ) 2 = 1 2 ( 2 γ ( s ) +2 γ ( t ) -2 c Y ( s,t ) ) ⟹ c Y ( s,t ) = γ ( s ) + γ ( t ) - γ ( s-t )
  3. It is 0 ≤ Var ( ∑ i=1 n a i Z( s i ) ) = ∑ i=1 n ∑ j=1 n a i a j c Y ( s i , s j ) = ∑ i=1 n ∑ j=1 n a i a j ( γ ( s i ) + γ ( s j ) - γ ( s i - s j ) ) = ∑ i=1 n a i γ ( s i ) + ∑ j=1 n a j γ ( s j ) - ∑ i=1 n ∑ j=1 n a i a j γ ( s i - s j ) hence ∑ i=1 n ∑ j=1 n a i a j γ ( s i - s j ) ≤ 0
