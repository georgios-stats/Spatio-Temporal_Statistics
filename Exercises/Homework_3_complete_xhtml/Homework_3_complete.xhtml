  2. Compute the block mean μ ( v ) for some block v=[ a,b ] ⊂ R .
  3. Compute the block covariance function c( v,s ) for some block v=[ a,b ] ⊂ R .
  4. Compute the block covariance function c( v,v' ) for some blocks v=[ a,b ] ⊂ R and v'=[ a',b' ] ⊂ R .
