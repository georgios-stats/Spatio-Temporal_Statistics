Exercise 2.
Let ( Z s ) s ∈ S be a specified statistical model. Assume that process ( Z s ) s ∈ S has known mean μ ( s ) = E ( Z( s ) ) and known covariance function c( ⋅ , ⋅ ) . Assume there is available a dataset { ( s i , Z i :=Z( s i ) ) } i=1 n . Assume that the matrix C such as [ C ] i,j =c( s i , s j ) has an inverse. Consider the “Kriging” estimator μ SK Consider the “Kriging” estimator Z SK ( s 0 ) of Z( s 0 ) at an unseen spatial location s 0 as the BLUE (Best Linear Unbiased Estimator) Z SK ( s 0 ) = w n+1 + ∑ i=1 n w i Z( s i ) = w n+1 + w ⊤ Z, for some unknown { w i } that we need to learn, and Z= ( Z 1 ,..., Z n ) ⊤ . Let w= ( w 1 ,..., w n ) ⊤ .
  1. Find sufficient conditions on w= ( w 1 ,..., w n ) ⊤ so that the Kriging estimator Z SK ( s 0 ) to be unbiased.
  2. Derive the MSE of Z SK ( s 0 ) as E ( Z SK ( s 0 ) -Z( s 0 ) ) 2 = w ⊤ Cw+c( s 0 , s ) -2 w ⊤ C 0 where C 0 is a vector such as [ C 0 ] i =c( s 0 , s i ) .
  3. Derive the Kriging estimator of Z( s 0 ) as Z SK ( s 0 ) = μ ( s 0 ) + C 0 ⊤ C -1 [ Z- μ ( s 1:n ) ] where μ ( s 1:n ) is a vector such as [ μ ( s 1:n ) ] i = μ ( s i ) .
  4. Compute the Kriging standard error σ SK = E ( Z SK ( s 0 ) -Z( s 0 ) ) 2 .
Solution.      The method is called Simple Kriging, and hence we denote it as SK.
  1. It is Z SK ( s 0 ) = w n+1 + ∑ i=1 n w i Z( s i ) = w n+1 + w ⊤ Z, where { w i } is a set of unknown weights to be learned.
    We assume that assume zero systematic error (unbiasness), hence E ( Z SK ( s 0 ) -Z( s 0 ) ) = E ( w n+1 + ∑ i=1 n w i Z( s i ) -Z( s 0 ) ) = w n+1 + ∑ i=1 n w i μ ( s i ) - μ ( s 0 ) which is satisfied given the assumption w n+1 = μ ( s 0 ) - ∑ i=1 n w i μ ( s i ) ⟺ w n+1 = μ ( s 0 ) - w ⊤ μ ( s 1:n ) where w= ( w 1 ,..., w n ) ⊤ .
  2. It is E ( Z SK ( s 0 ) -Z( s 0 ) ) 2 = Var ( Z SK ( s 0 ) -Z( s 0 ) ) = Var ( w n+1 + w ⊤ Z-Z( s 0 ) ) = Var ( w n+1 + w ⊤ Z ) + Var ( Z( s 0 ) ) -2 Cov ( w n+1 + w ⊤ Z,Z( s 0 ) ) = w ⊤ Cw+c( s 0 , s 0 ) -2 w ⊤ Cov ( Z,Z( s 0 ) ) = w ⊤ Cw+c( s 0 , s 0 ) -2 w ⊤ C 0 where C 0 = Cov ( Z,Z( s 0 ) ) , i.e. [ C 0 ] j =c( s j , s 0 ) .
  3. To learn the unknown weights { w i } we need to solve w SK = arg min w E ( Z SK ( s 0 ) -Z( s 0 ) ) 2 , subject to w n+1 = μ ( s 0 ) - w ⊤ μ ( s 1:n ) As E ( μ SK -Z( s 0 ) ) 2 does not depend on w n+1 we minimize 0 = ∇ w E ( Z SK ( s 0 ) -Z( s 0 ) ) 2 = ∇ w Var ( Z SK ( s 0 ) -Z( s 0 ) ) =2Cw-2 C 0 So I get w SK = C -1 C 0 So Z SK ( s 0 ) = w n+1 + C -1 C 0 Z = μ ( s 0 ) - ( C -1 C 0 ) ⊤ μ ( s 1:n ) + ( C -1 C 0 ) ⊤ Z = μ ( s 0 ) + C 0 ⊤ C -1 [ Z- μ ( s 1:n ) ]
  4. It is σ SK = E ( Z SK ( s 0 ) -Z( s 0 ) ) 2 = w SK ⊤ C w SK +c( s 0 , s 0 ) -2 w SK ⊤ C 0 = c( s 0 , s 0 ) - C 0 ⊤ C -1 C 0
