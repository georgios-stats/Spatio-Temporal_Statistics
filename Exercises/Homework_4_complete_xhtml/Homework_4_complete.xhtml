Exercise 1. ( ★ ) 
Show that the local characteristics Pr 1 ( x 1 | x 2 ) = 1 2 π exp ( - 1 2 ( x 1 - x 2 ) 2 ) Pr 2 ( x 2 | x 1 ) = 1 2 π exp ( - 1 2 ( x 2 - x 1 ) 2 ) do not define a proper joint distribution on R { 1,2 }
Solution.  
Definition. (Pseudo-likelihood) The pseudo likelihood pseudo- L( Z; θ ) of observables Z= ( Z 1 ,..., Z n ) ⊤ given parameters θ is an approximation of the likelihood L( Z; θ ) of observables Z= ( Z 1 ,..., Z n ) ⊤ given parameters θ which is equal to pseudo- L( Z; θ ) = ∏ i Pr ( Z i | Z -i , θ ) where Pr ( Y| θ ) is the joint pdf/pmf of the sampling distribution of Y given parameter θ .

Definition. (Maximum Pseudo-Likelihood Estimator) The Maximum Pseudo-Likelihood Estimator (MPLE) θ ˜ of θ is the maximizer of the pseudo likelihood function pseudo- L( Z; θ ) where the parameter θ is the argument and the observables Z= ( Z 1 ,..., Z n ) ⊤ are fixed values. θ ˜ = arg max θ ( pseudo- L( Z; θ ) )
